## 📖 Overview

This report consolidates **three complete digital forensic investigations** conducted as part of MSc Cybersecurity coursework.

Each case simulates a real-world cyber incident and follows a structured **DFIR (Digital Forensics & Incident Response)** methodology.

---

## 📁 Cases Included

### 🏦 Disk Forensics — Bank Fraud Case
- USB forensic image analysis  
- Password cracking (`langley`)  
- Steganography detection in PNG files  
- $5.55M suspicious transactions uncovered  

---

### 🌐 Network Forensics — Malware & C2 Analysis
- PCAP analysis using Wireshark  
- Agent Tesla / Raccoon Stealer identified  
- C2 servers mapped (`45.141.59.212`)  
- Data exfiltration via HTTP POST  

---

### 📄 Memory Forensics — Malicious PDF Case
- Memory dump analysis using Volatility  
- `AcroRd32.exe` identified as exploit vector  
- Fake processes (`expl0rer.exe`, `svch0st.exe`)  
- Credential theft + NTLM hashes extracted
